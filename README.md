## How to host website locally

### Cloning the repository
```bash
git clone https://github.com/kelanwu17/COSC3380-Library-Database.git
cd frontend
code .
```

### Deploying the website
```bash
cd frontend
npm install
npm start
```

### Deploying the backend/server
```bash
Backend:
cd backend
npm install
npm start
```
